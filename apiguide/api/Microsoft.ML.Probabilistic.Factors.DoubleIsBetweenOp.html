    The outgoing EP message to the isBetween argument.
    Remarks

    The outgoing message is a distribution matching the moments of isBetween as the random arguments are varied. The formula is proj[p(isBetween) sum_(x,lowerBound,upperBound) p(x,lowerBound,upperBound) factor(isBetween,x,lowerBound,upperBound)]/p(isBetween).

    Parameters
    Type Name Description
    Bernoulli isBetween

    Incoming message from isBetween.
    Gaussian X

    Incoming message from x.
    Gaussian lowerBound

    Incoming message from lowerBound.
    Gaussian upperBound

    Incoming message from upperBound.
    Returns
    Type Description
    Gaussian

    The outgoing VMP message to the lowerBound argument.
    Remarks

    The outgoing message is the exponential of the average log-factor value, where the average is over all arguments except lowerBound. Because the factor is deterministic, isBetween is integrated out before taking the logarithm.
